## CentOS7
备份配置文件

```
cp -a /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.bak
```

下载最新的repo镜像

```
wget -O /etc/yum.repos.d/CentOS-Base.repo https://mirrors.huaweicloud.com/repository/conf/CentOS-7-anon.repo
```

刷新缓存

```
yum clean all
yum makecache
```

更新软件包

```
yum update
```
