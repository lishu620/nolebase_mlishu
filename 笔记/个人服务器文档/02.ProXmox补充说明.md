## ProXmox VE 直通硬盘
### 全盘映射

使用PVE有时为了方便，需要将硬盘直通，一般有两种方式，一是硬件直通，一是全盘映射，这里介绍第二种（由于硬件直通需要把直通分组才能把按照PVE的系统盘隔离出来），方法如下：

#### 查看硬盘设备ID

```
ls /dev/disk/by-id
```

![[Pasted image 20250930232304.png]]

第一列就是硬盘设备ID

#### 硬盘映射

```
qm set 104 -sata1 /dev/disk/by-id/ata-WDC_WD10EZEX-22MFCA0_WD-WCC6YF25JY2P
```


> [!NOTE] 提示
> 104是需要映射给虚拟机的ID
> 然后-sata1是需要映射的虚拟机硬盘名
> 后面ata-XXX是硬盘ID

如果返回以下信息,说明已成功映射

```
update VM 104: -sata1 /dev/disk/by-id/ata-WDC_XXXX_XXXX_XXXX
```
