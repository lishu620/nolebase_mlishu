## 模拟机配置


> [!NOTE] 提示
> 如果使用压缩包而非VMDK控制器文件，直接从此操作

### 前置操作

连接到校园网之后：[点击下载](http://10.41.57.106/%E6%95%B0%E6%8D%AE%E5%AD%98%E5%82%A8/Huawei%20OceanStor%20eStor.zip)

完全解压系统后，打开模拟机系统

### 网络配置

查看VMware虚拟机中仅主机模式的子网网段，并合理分配IP
![[Pasted image 20251028101335.png]]
这里我将IP设定为59.168.1.100

由于所有网卡都是用`仅主机模式`，因此需要自行配置IP

![[Pasted image 20251021112815.png]]
### 序列号配置

选择序列号6，类型1或类型2

![[Pasted image 20251021112853.png]]

最后确认配置，输入`yes`

![[Pasted image 20251021112922.png]]
### 模拟器登录

安装过程中，可以直接登录查看。默认账号及密码：`admin / Admin@storage`

登录之后可以看到系统正在安装，根据硬件性能不同，时间⻓短不⼀（2min-15 min），暂时不要登录，等待约5分钟之后再登录

![[Pasted image 20251021113005.png]]

出现这个问题不用担心，点击重启客户机
![[Pasted image 20251028104018.png]]

只有进入这个界面，才算正式安装好

![[Pasted image 20251028105341.png]]

当出现`Old Password`证明安装完毕，输⼊原密码，修改新密码，这里设定为`Huawei12#$`

![[Pasted image 20251021114938.png]]

> [!ERROR] 警告
> 模拟环境不可挂起或快照

![[Pasted image 20251021115156.png]]

输入账号密码登录

![[Pasted image 20251021115250.png]]

## 重启后如何使用存储

在重启模拟器后，系统只会不会保留登录密码，登录密码为：`admin/Admin@storage`
![[Pasted image 20251028114906.png]]

输入旧密码和新密码就行
![[Pasted image 20251028114947.png]]

只要显示`admin:/>`就可以正常使用
