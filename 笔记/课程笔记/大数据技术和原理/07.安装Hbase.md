## 编辑配置文件
### hbase-env.sh

```
vi /opt/modules/hbase-1.2.6/conf/hbase-env.sh
```

在文件末尾添加以下文件

```
export JAVA_HOME=/opt/modules/jdk1.8.0_25
export HBASE_MANAGES_ZK=false
```

![](附件/图片/Pasted image 20251021145800.png)
### hbase-site.xml

```
vi /opt/modules/hbase-1.2.6/conf/hbase-site.xml
```

```
<configuration>
        <property>
                <name>hbase.zookeeper.quorum</name>
                <value>gyy1,gyy2,gyy3</value>
        </property>

        <property>
                <name>hbase.rootdir</name>
                <value>hdfs://gyy1:9000/hbase</value>
        </property>
        <property>
                <name>hbase.cluster.distributed</name>
                <value>true</value>
        </property>
        <property>
                <name>hbase.master.info.port</name>
                <value>60010</value>
        </property>
</configuration>
```

## 修改文件权限并分发

```
sudo chown -R hadoop:hadoop /opt/modules/hbase-1.2.6/
```

分发到其他节点

```
scp -r /opt/modules/hbase-1.2.6/ hadoop@gyy2:/opt/modules/
scp -r /opt/modules/hbase-1.2.6/ hadoop@gyy3:/opt/modules/
```

### 添加环境变量

```
sudo vi /etc/profile
```

```
export HBASE_HOME=/opt/modules/hbase-1.2.6/
export PATH=$PATH:$JAVA_HOME/bin:$HADOOP_HOME/bin:$HADOOP_HOME/sbin:$JRE_HOME/bin:$ZOOKEEPER_HOME/bin:$ZOOKEEPER_HOME/conf::$HBASE_HOME/bin
```

![](附件/图片/Pasted image 20251021151220.png)

```
sudo scp /etc/profile root@gyy2:/etc/
sudo scp /etc/profile root@gyy3:/etc/
```

```
source /etc/profile
```

## 启动

在任意一台主机上运行


> [!NOTE] 注意
> 启动顺序为：Hadoop > Zookeeper >Hbase


```
start-hbase.sh
```
