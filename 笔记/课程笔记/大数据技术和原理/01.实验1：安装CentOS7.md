## 系统配置
### 修改主机名
```
sudo hostnamectl set-hostname gyy1
```
### 修改泛域名解析文件
```
sudo vi /etc/hosts
```
#### /vi/hosts文件
```
192.168.78.100 gyy1
192.168.78.101 gyy2
192.168.78.102 gyy3
```
### 关闭防火墙
```
sudo systemctl stop firewalld.service
sudo systemctl disable firewalld.service
```
查看防火墙是否关闭
```
sudo systemctl status firewalld.service
```
![[Pasted image 20251117143908.png]]
### 修改网卡为静态
```
vi /etc/sysconfig/network-scripts/ifcfg-ens192
```
详细配置如下
```
TYPE="Ethernet"
PROXY_METHOD="none"
BROWSER_ONLY="no"
BOOTPROTO="none"
DEFROUTE="yes"
IPV4_FAILURE_FATAL="no"
IPV6INIT="yes"
IPV6_AUTOCONF="yes"
IPV6_DEFROUTE="yes"
IPV6_FAILURE_FATAL="no"
IPV6_ADDR_GEN_MODE="stable-privacy"
NAME="ens32"
UUID="42f94609-5d7b-4958-9d7f-f30546a0fd78"
DEVICE="ens32"
ONBOOT="yes"
IPADDR="192.168.78.100"
PREFIX="24"
GATEWAY="192.168.78.2"
DNS1="114.114.114.114"
IPV6_PRIVACY="no"
```

### 修改为命令行界面
查看当前登陆界面类型
```
runlevel
```
将登陆界面类型设置为命令行
```
systemctl set-default multi-user.target
```
同理可以将登录界面设置为图形化
```
systemctl set-default graphical.target
```
