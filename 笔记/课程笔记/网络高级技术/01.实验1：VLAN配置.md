## 基于接口划分VLAN

一般来说，对于VLAN的划分，使用trunk模式和access模式分割即可
### 拓扑图
![[Pasted image 20250917113810.png]]
### 配置代码
#### LSW1
```
system
vlan batch 10 20
int g0/0/1
port link-type access
port default vlan 10
int g0/0/2
port link-type access
port default vlan 20
int g0/0/3
port link-type trunk
port trunk allow-pass vlan 10 20
```
#### LSW2
```
system
vlan batch 10 20
int g0/0/1
port link-type trunk
port trunk allow-pass vlan 10 20
int g0/0/2
port link-type access
port default vlan 10
int g0/0/3
port link-type access
port default vlan 20
```

这样就完成了本实验中的VLAN10和VLAN20隔离

## Hybrid模式

华为VRP专属的接口模式
### 拓扑图
![[Pasted image 20250918110944.png]]

### 配置代码
#### LSW3
```
system
vlan batch 10 20 100
int g0/0/1
port link-type hybrid
port hybrid pvid vlan 10
port hybrid tagged vlan 10
port hybrid untagged vlan 10 100
int g0/0/2
port link-type hybrid
port hybrid pvid vlan 20
port hybrid tagged vlan 20
port hybrid untagged vlan 20 100
int g0/0/3
port link-type hybrid
port hybrid pvid vlan 100
port hybrid tagged vlan 100
port hybrid untagged vlan 10 20 100
```
