[[01.安装Rhel10]]
## 开机自动挂载安装镜像

编辑`fstab`文件

```
vi /etc/fstab
```

在最后一排添加这一排配置

```
/dev/sr0                /mnt/iso                iso9660 loop,ro         0 0
```

保存后测试

```
mount -a
```
## 关闭SeLinux
```
vi /etc/selinux/config
```

```
SELINUX=disabled
```
