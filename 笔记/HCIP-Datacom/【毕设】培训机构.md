# 培训机构售前/部署工程师（eNSP模拟）
## 一、公司背景

本公司需要将原有的培训机构内部网络改造，在改造完成后

### 公司环境调研

本次调研针对培训机构现有三栋建筑（公司大楼、培训大楼、公司附属酒店）展开，重点明确各建筑功能布局及网络改造相关规划，原网络方案为：公司大楼单独接入中国移动网络，培训大楼与公司附属酒店共同接入中国电信网络；本次改造核心为培训大楼部分区域功能调整，具体详情如下：

#### 一、培训大楼（核心改造区域）

本次改造重点优化6F-A区域功能，同时保留2F-A、5F-A教室功能，各楼层及房间用途、规格明确如下：

#### 二、公司附属酒店

酒店共6层，每层统一布局，具体为：8间标准房间 + 2间套房，无其他特殊功能区域。

大楼共5层，各楼层功能布局明确如下，无改造规划：

- 5F：财务部、设备管理科（两大部门集中布局）

## 配置代码

## 1.核心路由
### R1
```
undo t t
sys
sysn R1
int g0/0/0
	ip add 192.168.99.1 24
	vrrp vrid 1 virtual-ip 192.168.99.10
	vrrp vrid 1 priority 120
	quit
```
### R2
```
undo t t
sys
sysn R2
int g0/0/0
	ip add 192.168.99.2 24
	vrrp vrid 1 virtual-ip 192.168.99.10
	vrrp vrid 1 priority 100
	quit
```
### ER
```
vlan batch 99
int vlanif 99
	ip add 192.168.99.254 24
int g5/0/1
	po lin ac
	po de vl 99
int g5/0/2
	po lin ac
	po de vl 99
	quit
```
